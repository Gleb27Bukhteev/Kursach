# Восстановление пароля и SMTP

В проекте есть восстановление пароля через код.

Если SMTP не настроен, код показывается прямо на странице. Это сделано для учебного запуска, чтобы не обязательно подключать почту.

## Переменные для SMTP

```powershell
$env:SMTP_HOST = "smtp.example.com"
$env:SMTP_PORT = "587"
$env:SMTP_USER = "user@example.com"
$env:SMTP_PASSWORD = "password"
$env:SMTP_FROM = "user@example.com"
```

После этого запустить сервер:

```powershell
python -m uvicorn main:app --host 127.0.0.1 --port 8000 --reload
```

## Как работает

1. Пользователь нажимает "Забыли пароль?".
2. Frontend отправляет почту на `/api/auth/forgot-password`.
3. Сервер создает код на 15 минут.
4. Если SMTP настроен, код отправляется письмом.
5. Если SMTP не настроен, код возвращается в ответе для локального теста.
6. Пользователь вводит код и новый пароль.

## Важно

Для защиты проекта на реальном сервере нельзя показывать код восстановления в ответе. В учебной версии это нормально, потому что проект запускается локально.
