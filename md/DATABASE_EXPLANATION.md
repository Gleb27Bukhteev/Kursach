# База данных

Проект использует PostgreSQL и SQLAlchemy.

Настройки подключения находятся в `database.py`.

## Настройки по умолчанию

```text
host: 127.0.0.1
port: 5433
user: postgres
password: admin123
database: messenger_db
```

Если у вас другой порт или пароль, задайте переменные окружения:

```powershell
$env:POSTGRES_PORT = "5432"
$env:POSTGRES_PASSWORD = "ваш_пароль"
```

## Создание таблиц

Сначала должна существовать база `messenger_db`.

Создать таблицы:

```powershell
python create_db.py
```

При запуске приложения таблицы тоже создаются автоматически:

```powershell
python -m uvicorn main:app --reload
```

## Таблицы

```text
users
```

Пользователи. Хранит почту, никнейм, публичный ID, хеш пароля, аватар и данные для восстановления пароля.

```text
contacts
```

Контакты. Хранит связь "владелец контакта - добавленный пользователь".

```text
groups
```

Группы. Хранит название, аватар и создателя группы.

```text
group_members
```

Участники групп.

```text
messages
```

Сообщения. Хранит текст, отправителя, получателя или группу, файл/картинку, дату создания, редактирования, удаления и прочтения.

## Очистка данных

Удалить пользователей, группы, контакты и сообщения:

```sql
TRUNCATE TABLE messages, contacts, group_members, groups, users RESTART IDENTITY CASCADE;
```

Эта команда очищает данные, но не удаляет сами таблицы.

## Частые ошибки

Если проект не подключается к базе:

- проверьте, что PostgreSQL запущен;
- проверьте, что база `messenger_db` создана;
- проверьте порт: часто бывает `5432`, а в проекте по умолчанию `5433`;
- проверьте пароль пользователя `postgres`.
