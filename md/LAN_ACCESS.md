# Запуск в локальной сети

Это нужно, чтобы открыть мессенджер с телефона или другого компьютера в той же Wi-Fi сети.

## Быстрый способ

Запустить:

```powershell
.\start_lan.bat
```

Скрипт запустит сервер на:

```text
0.0.0.0:8000
```

И покажет IPv4-адрес компьютера.

На телефоне открыть:

```text
http://ВАШ_IPV4_АДРЕС:8000
```

Например:

```text
http://192.168.1.15:8000
```

## Ручной запуск

```powershell
python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

## Если не открывается

- телефон и компьютер должны быть в одной Wi-Fi сети;
- Windows Firewall должен разрешать Python для частных сетей;
- адрес надо открывать именно с IP компьютера, не `127.0.0.1`;
- сервер должен быть запущен с `--host 0.0.0.0`.
